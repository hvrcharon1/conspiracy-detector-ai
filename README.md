# 🕵️ Conspiracy Detector AI

An AI-powered conspiracy detection and personal safety intelligence system built with React and Claude AI.

## Features

- **6-Step Profile Builder** — Personal history, education, work history, workplace dynamics, incidents
- **Threat Level Analysis** — Scored 0–100 with LOW / MEDIUM / HIGH / CRITICAL classification
- **Conspiracy Timeline** — When and how the conspiracy began and evolved
- **Actor Mapping** — Who is involved, their role (Instigator / Active / Passive Enabler), and psychological drivers
- **Pattern Detection** — Career Suppression, Reputation Destruction, Immigration Theft, Authority Corruption, Psychological Warfare
- **Root Cause Analysis** — Why this happens (envy, casteism, insecurity, territorial behavior, corrupt systems)
- **Vulnerability Assessment** — What unintentionally enabled the conspiracy
- **4-Part Protection Plan** — Immediate / Legal / Reputation / Psychological actions

## Tech Stack

- React (JSX)
- Anthropic Claude API (`claude-sonnet-4-20250514`)
- Tailwind CSS utility classes
- No external dependencies beyond React

## Use Cases

- Workplace conspiracy and mobbing detection
- Cross-border corporate power structure analysis
- Casteism / religious favoritism identification
- Immigration opportunity theft scenarios
- Corrupt authority involvement patterns

## Setup

1. Clone this repository
2. Deploy the `.jsx` file in any React environment (Vite, CRA, Next.js)
3. Ensure the Anthropic API key is configured in your environment
4. Run and open in browser
